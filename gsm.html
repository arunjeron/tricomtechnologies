GSM SMS Terminals

WINCELL GSM Fixed Cellular Terminal (FCT) converts GSM/CDMA phone to copper wire phone for easy interface with appliances like Telephone instruments, EPABX, Public call office billing machine etc. GSM/CDMA - FCTs prevents all the health hazards associated with GSMCDMA mobile phones and the inconveniences. It gives the comforts of an ordinary telephone instruments and allows you to use external gain antenna for indoor use with out any boosters/repeaters. WINCELL has an option for inbuilt auto router for least cost routing to save costs. The enhanced least cost routing facility, routs the call through the most economical network (Land line or GSM/CDMA network). External CDMA FWTs can also be used as land line for economical call routing. Optional GPRS Port for PC connectivity.

Feature

  • Easy connectivity to PABX/Phone through RJ-11 connector.
  • Auto routing (Least Cost) option between GSM and PSTN / CDMA lines.
  • User configurable tariff entry options.
  • Multiple SIM card/handsets support in Multi line FCT.
  • Least cost routing algorithm option across four SIM cards/handsets.
  • Injects CLI in line for PABX junctions.
  • Single GPRS port option in multi line FCT.
  • Call success rate 100%.
  • Compatible to all types of PCOs/STD machines.
  • Compatible to all types of PABX.
  • Option for 2 GSM and 2 CDMA in one Multi line FCT.
